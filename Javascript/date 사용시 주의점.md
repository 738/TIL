# Javascript에서 Date 사용할 때 주의할 점

## 2018.11.03

### setMonth

```javascript
let a = new Date(‘2018-10-31');    // 10월 31일로 세팅
a.setMonth(10);                    // 11월로 세팅
// a는 12월 1일이 됨
// 그 이유는 10월 31일을 11월로 세팅하면 11월 31일이 되는데
// 11월에는 31일이 없으므로 12월 1일이 됨
a.setMonth(10, 1);        // 이런 식으로 date도 줘야할듯
```
